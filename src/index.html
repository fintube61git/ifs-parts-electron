🎯 Quick Start

This application automatically loads images and questions on startup.

  1. Start Reviewing: Use ← Prev / Next → to navigate between images and answer questions.
  2. Answering: Fill out the questions for each image. Your responses are automatically saved.

🎮 Using the Application

  • Navigation: Use ← Prev / Next → buttons or arrow keys to move between cards.
  • Zoom: Use +/- buttons or "⊞" to fit image to screen.
  • Clearing Responses: Use "Clear Current" or "Clear All" to erase responses.

💾 Export Options

  • Export Browser File: Creates a viewable HTML file with all your responses.
